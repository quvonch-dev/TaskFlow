# ✦ TaskFlow — Todo Manager

Zamonaviy, tez va chiroyli vazifalar menejeri. Sof **HTML, CSS va JavaScript** bilan yozilgan — hech qanday framework kerak emas.

![TaskFlow Preview](https://via.placeholder.com/800x400/0e0f11/7c6ef5?text=TaskFlow+Todo+Manager)

---

## 🚀 Xususiyatlar

- ✅ **Vazifa qo'shish** — matn, ustuvorlik, kategoriya va muddat bilan
- 🏷️ **Kategoriyalar** — Ish, Shaxsiy, Salomatlik, O'rganish
- 🔴 **Ustuvorlik darajalari** — Past / O'rta / Yuqori
- 📅 **Muddat belgilash** — muddati o'tgan vazifalar alohida ko'rsatiladi
- 🔍 **Filtrlash** — Barchasi / Faol / Bajarilgan
- ✏️ **Inline tahrirlash** — vazifa matnini to'g'ridan-to'g'ri bosib tahrirlang
- 📊 **Progress bar** — kunlik maqsad kuzatuvi
- 💾 **LocalStorage** — sahifani yangilasangiz ham ma'lumotlar saqlanadi
- 📱 **Responsive dizayn** — mobil va desktop uchun

## 🛠️ Ishga tushirish

1. Repozitoriyni klonlang:
   ```bash
   git clone https://github.com/USERNAME/taskflow.git
   ```

2. Papkaga o'ting:
   ```bash
   cd taskflow
   ```

3. `index.html` faylini brauzerda oching yoki VS Code Live Server ishlatib ko'ring.

> Hech qanday `npm install` yoki build qadami kerak emas!

---

## 📦 GitHub Pages orqali joylashtirish

1. GitHub'da yangi repo yarating
2. Fayllarni push qiling:
   ```bash
   git init
   git add .
   git commit -m "feat: initial TaskFlow release"
   git branch -M main
   git remote add origin https://github.com/USERNAME/taskflow.git
   git push -u origin main
   ```
3. **Settings → Pages → Branch: main** ni tanlang
4. Bir necha daqiqadan so'ng saytingiz tayyor: `https://USERNAME.github.io/taskflow`
